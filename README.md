# share_beautifulsoup_docset

[![Release](https://github.com/scillidan/share_beautifulsoup_docset/actions/workflows/release.yml/badge.svg)](https://github.com/scillidan/share_beautifulsoup_docset/actions/workflows/release.yml)

This repo is for generating docset files from [crummy/BeautifulSoup](https://github.com/crummy/BeautifulSoup). For using in [Zeal](https://zealdocs.org), [Dash](http://kapeli.com/dash), [Velocity](https://velocity.silverlakesoftware.com/) ...

## Usage

- Zeal → Tools → Docsets → Add feed
	- `https://raw.githubusercontent.com/scillidan/share_beautifulsoup_docset/refs/heads/master/Beautiful_Soup_4.xml`
